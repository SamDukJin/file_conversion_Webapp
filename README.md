# File Converter Webapp

This website application is a modern, fast, and private **client-side** only file conversion tool that built with React, TypeScript, and Vite. This Web application allows user to convert many file formats to PDF entirely with configurations. All-in-one in your browser with no server-side processing, allowing full privacy.

## Features

- **Privacy First:** All conversions happen locally in your browser. No sign-in no server downloading your file, just pure client-side processing.
- **HTML/Markdown to PDF:** Convert rich text and documentation to professional PDFs.
- **Image to PDF:** Effortlessly bundle images into PDF documents.
- **Drag & Drop:** Intuitive interface for easy file handling.
- **Live Preview:** See your files before you convert them.
- **Customization:** Adjust page sizes (A4, Letter), margins, and quality.

## Tech Stack

- **Framework:** [React 18](https://reactjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Bundler:** [Vite](https://vitejs.dev/)
- **PDF Generation:** [jsPDF](https://github.com/parallax/jsPDF)
- **HTML Rendering:** [html2canvas](https://html2canvas.hertzen.com/)
- **Archive Handling:** [JSZip](https://stuk.github.io/jszip/)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SamDukJin/file-conversion-Webapp.git
   ```

2. Navigate to the project directory:
   ```bash
   cd file-conversion-Webapp
   ```

3. Install dependencies:
   ```bash
   npm install
   ```
